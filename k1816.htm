K1816

The K1816, KM1816, KR1816, KS1816 series, manufactured using proportionally scaled pMOS technology and
intended for use as microcontrollers that require short programs,
a small amount of memory for variables and limited input / output information.
It include the following types:
K1816VE31, KM1816VE31, KR1816VE31 - single-chip 8-bit microcomputer without internal ROM;
KM1816VE35, KR1816VE35 - single-chip 8-bit microcomputer without internal ROM;
KR1816VE39 - single-chip 8-bit microcomputer without internal ROM;
KM1816VE48, KR1816VE48 - single-chip 8-bit microcomputer with PROM with UV erasure;
KM1816VE49 - single-chip 8-bit microcomputer with mask ROM;
KM1816VE51 - single-chip 8-bit microcomputer with mask ROM;
KM1816VE751, KS1816VE751 is a single-chip 8-bit microcomputer with PROM with UV erasure.
The ICs of the K1816, KM1816, KR1816 and KS1816 series have an identical structure,
but differ from each other only in the presence or absence of internal program memory,
the amount of internal RAM, the type and size of ROM, and speed.
For example, KM1816VE48 contains an EPROM, in which information is erased by ultraviolet irradiation,
that is, the user can independently write information to program memory, in contrast to KR1816VE49,
where this operation is carried out using a separate template during its manufacture.

K1818

KL1818, KM1818, KR1818, KS1818 series, manufactured using TTLSh or CMOS technology, includes the following types:
KR1818VA19 - universal asynchronous transceiver (UART);
KM1818VV1, KR1818VV1 - serial-parallel interface;
KM1818VV5, KR1818VV5 - parallel-serial interface
KM1818VV52 - communication controller;
KM1818VV61 - 8-bit serial communication controller (without exchange protocols);
KM1818VG01 - two-channel multifunctional serial controller;
KL1818VG62 - interrupt control circuit;
KP1818VG63 - Video controller;
KL1818VG65 - PC control logic;
KL1818VG79 - control circuit of the trunk (interface);
KL1818VG80 - microprocessor interface control circuit;
KR1818VG93 - programmable floppy disk controller;
KR1818VZh1 - programmable cyclic oscillator-controller (circuit for detecting single and group errors);
KM1818VI3, KR1818VI3 - address marker detector;
KM1818VI13 - 5-channel programmable timer;
KR1818VK12 - modified frequency-modulated code generator and universal interrupt circuit;
KR1818VN19 - universal programmable interrupt controller;
KM1818VM01, KR1818VM01 - single-chip 8-bit microprocessor (TTLSh);
KM1818VT1 - desktop computer interface with external memory devices;
KM1818VF4, KR1818VF4 - cyclic redundancy code generator;
KM1818VF12 - modified frequency-modulated code generator and universal interrupt circuit;
KM1818PTs1, KS1818PTs1 - 20-channel frequency divider with 16 programmable division ratios;
KR1818PTs2 - programmable frequency divider;
KR1818PTs3 - programmable frequency divider;
KR1818PTs4 - programmable frequency divider.
